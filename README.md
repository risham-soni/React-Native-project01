# 📱 Profile Card App

A simple and clean **Profile Card App** built using **React Native with Expo**.  
This app displays my profile information including image, name, profession, and social media links.

---

## 🚀 Features

- 👤 Profile Image Display
- 📝 Name & Profession
- 🔗 Clickable Social Links:
  - GitHub
  - LinkedIn
  - LeetCode
- 📱 Responsive UI Design
- 🎨 Clean and Simple Layout

---

## 🛠️ Built With

- React Native
- Expo
- JavaScript
- StyleSheet API
